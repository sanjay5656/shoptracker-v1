# ShopTracker V1

Smart inventory and profit tracking for small retail shops in India.

## The Problem
Local shop owners track sales in rough notebooks.
They don't know their daily profit, which category sells best,
or how much they lose to bargaining every day.

## The Solution
Three simple actions:
- **BUY** — Record wholesale purchase (30 seconds)
- **SELL** — Record sale with bargain tracking (10 seconds)  
- **SEE** — Automatic dashboard, zero manual work

## Unique Feature
**Bargain Price Tracking** — Records the difference between
asking price and actual received price. No other app in India tracks this.

## Tech Stack
- Backend: Python + FastAPI + SQLite
- Frontend: HTML + CSS + JavaScript (mobile-first)
- Deploy: Railway.app
